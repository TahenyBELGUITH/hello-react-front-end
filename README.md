# Hello Rails Back End

 Create a React front-end with a Rails back-end and connect them to display a random greeting message.

## Made With

- Ruby on Rails

## Link to BackEnd App
[link](https://github.com/TahenyBELGUITH/hello-rails-back-end)

## Getting Started

### Setup

To setup the project locally: clone the repository using:

```
git clone https://github.com/TahenyBELGUITH/hello-rails-back-end.git
```

### Install dependencies
run:
```
bundle install
```

### Setup database
run:
```
rails db:setup
rails db:migrate
rails db:seed
```

### Run the app
run:
```
rails s
```
